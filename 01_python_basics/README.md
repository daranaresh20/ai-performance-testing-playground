# Phase 1: Python Basics for Performance Engineers

These scripts help you apply your existing performance testing knowledge using Python 3.12.

## Scripts
- `response_time_stats.py` — Calculate min, max, avg, p90, p95, p99 from a list of response times
- `csv_report_parser.py` — Parse LoadRunner/JMeter CSV exports using pandas

## Key Concepts
- Working with lists and dicts (response time data)
- Reading/writing CSV files (test result reports)
- Basic statistics (percentiles)
- `rich` library for pretty terminal output
