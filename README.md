# 🧪 AI Performance Testing Playground

> A hands-on learning repo for **AI-powered Performance Testing** — combining 7+ years of performance engineering expertise with modern AI/LLM testing, Python automation, and SRE observability practices.

---

## 🎯 Goal
Bridge traditional performance testing (LoadRunner, RTE) with AI/GenAI testing concepts to transition into **DevOps / SRE roles**.

---

## 📁 Repo Structure

```
ai-performance-testing-playground/
├── 01_python_basics/          # Python scripts for perf testing fundamentals
├── 02_api_load_testing/       # HTTP load testing with Python (requests, aiohttp)
├── 03_ai_response_testing/    # Validate AI/LLM API responses with Python
├── 04_llm_benchmarking/       # Benchmark LLM latency, throughput, token speed
├── 05_genai_test_automation/  # Auto-generate test cases using OpenAI API
├── 06_observability_sre/      # SRE concepts: SLI, SLO, error budgets, alerting
├── 07_github_actions_ci/      # CI/CD pipelines for automated test runs
└── resources/                 # Links, notes, study materials
```

---

## 🚀 Getting Started

### Prerequisites
- Python 3.12+
- pip
- Git

### Setup
```bash
git clone https://github.com/daranaresh20/ai-performance-testing-playground.git
cd ai-performance-testing-playground
pip install -r requirements.txt
```

### Run your first test
```bash
python 02_api_load_testing/basic_load_test.py
```
