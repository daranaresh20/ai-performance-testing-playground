# Phase 5: GenAI Test Automation

Use AI to **generate test cases automatically** — the next evolution of test automation.

## Concept
Instead of manually writing test cases, prompt an LLM to generate them for you:
1. Describe the API/feature under test
2. LLM generates test scenarios, edge cases, and expected behaviors
3. You validate and execute them with your Python scripts

## Files
- `generate_test_cases.py` — Uses an LLM to generate test cases for a given API endpoint

## Why This Matters for Your Career
- Shows AI + QA integration skills
- Directly relevant to DevOps/SRE automation pipelines
- Reduces manual test design effort significantly
