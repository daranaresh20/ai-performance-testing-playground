# Phase 7: GitHub Actions CI/CD

Automate your performance test runs using GitHub Actions — this is a core DevOps/SRE skill.

## What's in `.github/workflows/`
- `run_perf_tests.yml` — Auto-runs your Python performance tests on every push

## What This Demonstrates
- CI/CD pipeline setup (key DevOps skill)
- Automated test execution without human intervention
- Test results as artifacts stored in GitHub

## How to Enable
The workflow in `.github/workflows/run_perf_tests.yml` will automatically run when you push to `main`.

Check the **Actions** tab in your GitHub repo to see results!
